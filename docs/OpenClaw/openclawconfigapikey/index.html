OpenClawжҺҘе…Ҙе№іеҸ°API Key

жң¬ж•ҷзЁӢе°ҶиҜҰз»ҶжҢҮеҜјдҪ еҰӮдҪ•е°ҶOpenClawжҺҘе…ҘйҳҝйҮҢдә‘зҷҫзӮје№іеҸ°пјҢеҢ…жӢ¬иҺ·еҸ–API KeyгҖҒй…ҚзҪ®жЁЎеһӢзӯүе®Ңж•ҙжӯҘйӘӨгҖӮ

йҖүжӢ©й…ҚзҪ®ж–№ејҸ

ж №жҚ®дҪ зҡ„и®ҝй—®ж–№ејҸпјҢйҖүжӢ©дёӢж–№еҜ№еә”зҡ„й…ҚзҪ®ж ҮзӯҫйЎөпјҡ

  • жң¬ең°й…ҚзҪ®пјҡзӣҙжҺҘеңЁи®ҫеӨҮжң¬ең°жөҸи§ҲеҷЁи®ҝй—®пјҢж— йңҖй…ҚзҪ®SSHйҡ§йҒ“е’ҢзҪ‘е…ід»ӨзүҢ
  • иҝңзЁӢй…ҚзҪ®пјҡйҖҡиҝҮWindowsжөҸи§ҲеҷЁиҝңзЁӢи®ҝй—®пјҢйңҖиҰҒй…ҚзҪ®SSHйҡ§йҒ“е’ҢзҪ‘е…ід»ӨзүҢ

дёҖгҖҒйҳҝйҮҢдә‘зҷҫзӮје№іеҸ°вҖӢ

1.1 зҷ»еҪ•йҳҝйҮҢдә‘вҖӢ

и®ҝй—®йҳҝйҮҢдә‘е®ҳзҪ‘пјҡйҳҝйҮҢдә‘-и®Ўз®—пјҢдёәдәҶж— жі•и®Ўз®—зҡ„д»·еҖј

дҪҝз”Ёд»ҘдёӢд»»дёҖж–№ејҸзҷ»еҪ•пјҡ

  • йҳҝйҮҢдә‘иҙҰеҸ·

  • й’үй’ү

и®ҝй—®ең°еқҖпјҡеӨ§жЁЎеһӢжңҚеҠЎе№іеҸ°зҷҫзӮјжҺ§еҲ¶еҸ°

1.3 иҺ·еҸ–API KeyвҖӢ
